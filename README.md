# W365toFET

```
W365toFET path/to/sp001.json

    -> path/to/sp001.fet
```

Die Stundenplan-Daten werden von der JSON-Datei eingelesen.

Ausgegeben wird eine FET-Datei im selben Ordner.

## Aktueller Stand (11.11.2024)
